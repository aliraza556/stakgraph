string

/map

Generate a text-based map of node relationships

query Parameters
node_type
string
Enum: "Page" "Function" "Class" "Trait" "Datamodel" "Request" "Endpoint" "UnitTest" "IntegrationTest" "E2etest"
Example: node_type=Page

name
string
Example: name=LeaderboardPage

Name of the starting node

ref_id
string
Example: ref_id=page_123

Reference ID of the starting node

tests
boolean
Default: true

Include test files in the map

depth
integer
Default: 7

Maximum depth of relationships to traverse

direction
string
Default: "both"
Enum: "incoming" "outgoing" "both"

Direction of relationships to traverse

trim
string
Example: trim=File,Directory

Comma-separated list of node types to trim from the map

Responses

Local development server

http://localhost:3000/map
